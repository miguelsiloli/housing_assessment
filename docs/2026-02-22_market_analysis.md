## Appendix: Scoring Methodology

### Value Score Formula
```
Base Score = (50 - price_per_m2)
+ 8  if furnished
+ 5  if balcony
+ 7  if terrace
+ 6  if AC
+ 4  if elevator
+ 10 if parking
+ 5  if renovated
+ 8  if bills included
+ size bonus (1-5 points)
+ typology bonus (T2: +5, T3: +8)
```

### Distance Score Formula
```
work_score = max(0, 35 - (distance_km × 3.5))      # Max 35 points
school_score = max(0, 15 - (distance_km × 1.5))    # Max 15 points
distance_score = work_score + school_score         # Max 50 points
```

### Total Score
```
total_score = value_score + distance_score
```

**Note:** Apartments without geocoded addresses receive distance_score = 0, which may artificially lower their ranking.
